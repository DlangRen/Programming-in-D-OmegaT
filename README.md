# 欢迎来到 Programmin-in-D-OmegaT 项目

### 目标：
  用 OmegaT 软件辅助翻译 Programming-in-D。

### 项目源由：

翻译时，常想边看着原文边编辑，有时还想来个“机器翻译”提示，自动显示词汇表、模糊匹配等等。多人翻译时，大家的成果都可自动显示在自己电脑上，这些，都可通过 OmegaT 很简单的实现。
